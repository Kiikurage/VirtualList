# 仮想リストUI

非常に多くの要素を含む、長いスクロール領域を持ったUIは描画処理に時間がかかる。

実際に画面上に見えている要素だけを描画することで、パフォーマンスを大幅に向上させることができる。

このリポジトリでは、[React](https://react.dev/)での仮想リストの実装方法について検討する。
